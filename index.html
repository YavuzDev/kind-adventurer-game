Progress

Blender

I used Blender to create the 3D Models for the game. When comparing the first model I made to the latest models there is a clear improvement. I think I really improved technically and creatively with Blender and creating 3D models.

Pixel Art

I used Pixelorama to create Pixel Art. The first sprite I made was quite bad, it did not look how I imagined it all. But when comparing it to the Player Sprite, which also has an attack animation, there is a big improvement.

3D Game Development

As for the actual development of the game, I used Godot. Godot is a game engine used for both 2D and 3D games. In the first GIF you can see some prototypes I made to test out some aspects of the game engine. In the second GIF you can see that I was able to make an actual game starting from the prototypes. I think I was able to create a good game without any knowledge about game development.
